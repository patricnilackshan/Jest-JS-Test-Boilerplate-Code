# JEST - JS TESTING

### 1. Initialize a node project

```shell
npm init -y
```

### 2. Install JEST as DevDependency

```shell
npm i --save-dev jset
```

### 3. Modify 'test' value to "jest" or "jest --coverage" for detailed report
```
  "scripts": {
    "test": "jest --coverage"
  },
```

### 4. Create `*.test.js` scripts

### 5. `npm test` to test the scripts.
